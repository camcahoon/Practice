List stuff in current dirctory:
ls
ls -la (long list and lists hidden files)

Make new directory:
mkdir [dirctory name]

Change directories:
cd [directory name]

Setup go in a dirctory:
go mod init github.com/camcahoon/practice (example)

Push to a repo:
git add [file name]
git commit
git push

Set up a binary/build thing:
go build

Use the binary:
./[binary name]
